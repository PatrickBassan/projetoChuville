# Projeto Chuville

## Descrição
A aplicação Chuville tem o objetivo de facilitar o monitoramento e prevenção contra alagamentos, utilizando sensores de profundidade em rios em conjunto com arduino.

## Stack/Tecnologias
- Front-end: React
- Back-end: PHP, Python
- Banco de dados: MySQL
- Servidor: AWS
